How to run the example
=======================


1- Modify the Twittercredentials.py file to update Twitter API credentials. (the credentials will be sent via email).
2- type: cd extweetwordcount
3- type: sparse run
4- Either after you stop after a while or in a separate window,
type: python histgram.py 10,50
5- Either after you stop after a while or in a separate window,
type: python finalresults.py
